# Keyboard Shortcuts 

- [Windows](#windows)
- [MacOS](#mac)

## Windows

### 🖥 OS
- Switch Application: `Alt` + `Tab`

### 👩‍💻 JetBrains IDE's

- Code Folding: `Ctrl` + `.`
- Search Everywhere: `Ctrl` + `Shift` + `F`
- Search File Names: `Shift` * 2
- Search File: `Ctrl` + `F`
- Replace Text: `Ctrl` + `R`

### 🌐  Google Chrome
- Refresh: `Ctrl` + `R`
- New Tab: `Ctrl` + `T`

### 📹 Zoom
- Toggle Audio: `Alt` + `A`
- Toggle Video: `Alt` + `V`
